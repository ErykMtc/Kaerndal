Grid section

Grid section help you to build layout with a lot of shapes. Our Grid uses 12 column system, divide him if you want

#Row

Row is kind a container for columns. We can visualizate it like a box, where we put bricks but the maximum size is 12

#Columns

Columns are the section where we can put our content and we can divide our page to few size section
